Flujo sugerido

Del código local a un bucket público en muy pocos pasos

Paso 1

Preparar archivos estáticos

Trabaja tu landing en local o en CloudShell y deja listos tus archivos HTML, CSS, imágenes y recursos.

Paso 2

Subir al bucket

Copia el contenido al bucket de Cloud Storage y define `index.html` como documento principal del sitio.

Paso 3

Compartir la URL

Habilita acceso público según el escenario del curso y publica una URL básica para mostrar el resultado.

Idea clave

Cloud Storage no solo guarda archivos: también puede ser el punto de partida perfecto para publicar una web educativa.

Para un hosting estático básico, la experiencia es directa, limpia y muy valiosa para entender cómo GCP resuelve necesidades reales sin sobrearquitectura.
